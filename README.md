# AI Resume Analyzer Agent

## Project Overview

This project is an AI-powered Resume Analyzer Agent.

The system reads a PDF resume and extracts text for further analysis.

## Features Completed (Day 1)

- PDF Upload
- PDF Text Extraction
- Resume Statistics
- Basic Skill Detection
- Text Export

## Technologies Used

- Python
- Google Colab
- PyPDF2

## Upcoming Features

- ATS Score Generation
- AI Resume Analysis
- Job Recommendation Engine
- Interview Question Generator
- Cover Letter Generator
